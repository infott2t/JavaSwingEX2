# JavaSwingEX2

이 프로젝트는 정보 시스템을 구축하기 위한 CRUD 기능을 제공합니다. 주요 기술 스택으로는 Java SE와 MySQL을 사용하였으며, 관리자 기능을 포함하고 있습니다.

아직 다 완성하지는 않았고, 전의 이 프로젝트를 대신하고, 또 여기의 글들을 참조해보세요.
